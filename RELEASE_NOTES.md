# GoodbyeDPI Turkey Interface

Windows icin hazir masaustu arayuzu.

## Indirme

- `GoodbyeDPI Turkey Interface_x.y.z_x64-setup.exe`

## Kurulum

1. Kurulum dosyasini indirin.
2. Uygulamayi acin.
3. Windows UAC penceresi gelirse yonetici izni verin.
4. `Baslat` ile varsayilan metodu calistirin.

## Ne Yapar

- GoodbyeDPI Turkey release ZIP'indeki komutlari hazir preset olarak sunar
- Arka planda calisabilir ve sistem tepsisine inebilir
- Istenirse Windows acilisinda otomatik baslayabilir

## Notlar

- Uygulama yonetici izni ister.
- Antivirus yazilimlari `WinDivert` dosyalarina mudahale edebilir.
- Sorun yasarsaniz once `Varsayilan`, sonra alternatif presetleri deneyin.

## Bilinen Durum

- Discord ve benzeri uygulamalarda sonuc ag yapisina gore degisebilir.
- Build dosyalari guncellenirken aktif `WinDivert` surucusu kapatilmalidir.
